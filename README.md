# 2022 Project
## 목표
* 적용해보고 싶은 라이브러리의 적용 또는 시도해보고 싶었던 간단한 프로젝트를 진행하는 것을 목표로 한다.
* 최소 2주에 1개 이상의 결과물을 업로드 하는 것으로 목표로 한다.


## 개발 환경 설정
### 의존성 설치 및 실행 방법
* node 및 npm version
```text
node v14.18.2
npm 6.14.15
```

* 의존성 설치
```text
npm i 또는 npm install
```

* 로컬 서버 실행
```text
npm start
```
